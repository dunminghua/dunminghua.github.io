977. Squares of a Sorted Array

Posted on 2021-12-09 Edited on 2021-12-18 In Algorithms Views: Valine:
题目：给你一个按非递减顺序排序的整数数组nums，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。

总结

  • method 2 和 1 属于暴力破解法，对数组平方后用快速排序排序, 时间复杂度是 $O(n + n/log(n))$
  • method 3 使用双指针法，从 input 左右两边向中间逼近，时间复杂度 $O(n)$, 相较于前两种方法效率高不少。
  • sorted(list) VS. list.sort()
    1. sorted() worked on any iterable, not just lists, Strings, tuples, dic, generators, etc., which return a new sorted list, leaving the original list unaffected.
    2. use sort() when you want to mutate the list.
    3. list.sort() is faster than sorted() bacause it doesn’t have to creat a copy
