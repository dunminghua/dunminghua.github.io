Working with multiple windows in vim

Posted on 2022-04-19 In Tools Views: Valine:

你是否会因为同时打开多个代码文件导致显示不全而烦恼？此文罗列了一些有用的操作！

prefix键 <Ctrl + w>

  1. Opening split windows
命令 操作
Ctrl+w s 水平分割同一个文件
Ctrl+w v 垂直分割同一个文件
:sp[lit] filename 水平分割窗口并读取filename
:vsp[lit] filename 垂直分割窗口并读取filename
  1. Closing split windows
命令 操作
:q[uit] 关闭当前的window
:on[ly] 关闭除当前active window的其他所有窗口
  1. Changing focus between windows
命令 操作
ctrl+w w 循环切换文件
ctrl+w h 切换光标到左边文件
ctrl+w j 切换光标到下方文件
ctrl+w k 切换光标到上方文件
ctrl+w l 切换光标到右方文件
  1. resizing windows
命令 操作
ctrl+w + 增加高度1行
ctrl+w - 减少高度1行
ctrl+w _ 最大化当前窗口的高度
ctrl+w | 最大化当前窗口的宽度(超有用)
ctrl+w = 将所有窗口恢复成平均分割的状态
  1. Moving windows
命令 操作
ctrl+w r Rotate all windows(旋转所有窗口)
ctrl+w x Exchange current window with its neighbor(交换相邻窗口)
ctrl+w H move current window to far left(移至最左边)
ctrl+w J Move current window to bottom(移至底部)
ctrl+w K Move current window to top(移至顶部)
ctrl+w L Move current window to far right(移至最右边)

see more :help ctrl+w
